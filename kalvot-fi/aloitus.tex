\emph{Matematiikan opettajankin} on hyvä tuntea \LaTeX{}: se

\emph{Avainsanoja:} markup languages, GNU ja vapaa lähdekoodi 
\end{slide}


\begin{slide}{Mikä \LaTeX{} on?}
\LaTeX{}, tarkemmin \LaTeXe{} on taitto-ohjelma, jolla voi tehdä
rakenteisia dokumentteja, esimerkiksi

\begin{itemize}
\item kirjoja,
\item opintomonisteita,
\item tenttikysymyspapereita,
\item Pro gradu -tutkielman tai
\item esitysgrafiikkaa --- kuten nyt.
\end{itemize}
\end{slide}


\begin{slide}{Mikä \LaTeX{} on?\dots}
Rakenteisuus tarkoittaa, että käsikirjoituksessa (eli lähdekoodissa) on
\begin{itemize}
\item loogisia (\verb|\begin{document}|) ja 
\item typografisia (\verb|\textbf{lihavoituna}|)
\end{itemize}
komentoja sekä leipätekstiä. Lähdekoodin voi kääntää eri muotoihin,
joista yleisimmät ovat
\begin{itemize}
\item PDF (Portable Document Format) ja
\item HTML (HyperText Markup Language)
\end{itemize}
\end{slide}


\begin{slide}{Kurssi on hyödyllinen, jos}

\item avoimen lähdekoodin ohjelmia.
\end{itemize}
\end{slide}
